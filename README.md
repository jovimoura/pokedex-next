## 🇺🇸 If you want to analyze the project on your machine, follow this step by step: 😁

<br>

### Clone😎

```
git clone https://github.com/jovimoura/pokedex-next
```

### Access the folder🤓

```
cd <nome-da-pasta>
```
### Install dependencies🤠
```
npm install
```
### Start application🤩
```
npm run serve
```
### default port:🤗

```
http://localhost:3000/
```
